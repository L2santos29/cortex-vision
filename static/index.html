📖 About Cortex-Vision

🧠 What is Cortex-Vision?

A production-oriented object recognition platform powered by YOLOv8. It processes images, videos, and live webcam streams, rendering bounding boxes with confidence scores in real time.

⌨️ Keyboard Shortcuts

Space — Toggle webcam
E — Export CSV

1–5 — Switch tabs (Images, Video, Webcam, History, About)

🛡️ Security Notes

All uploads are processed in memory and discarded after analysis. Your images and videos are never stored permanently on the server.

Disclaimer: This is a demonstration project. For production use, deploy behind a reverse proxy with HTTPS and configure a strong API_KEY via environment variable.
