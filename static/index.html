📖 About Cortex-Vision

🧠 What is Cortex-Vision?

Cortex-Vision is an object recognition platform powered by YOLOv8, one of the fastest and most accurate real-time object detection models available. It combines computer vision with a modern web interface to make image analysis accessible to everyone.

🎯 What is it for?

Upload images or videos, or use your webcam — Cortex-Vision detects and classifies objects in real time. Each detected object is highlighted with a colored bounding box showing its class label and confidence score. Use it for:

📸 Batch image analysis — process hundreds of photos at once
🎬 Video analysis — extract frames and detect objects throughout a video
📹 Live webcam — real-time detection at 1 frame per second
📊 Data export — download results as CSV or annotated images

⚙️ How does it work?

1. Upload — images, video, or webcam feed are sent to the server
2. Inference — YOLOv8 (Ultralytics) processes each frame in memory
3. Detection — objects are identified with bounding boxes and confidence scores (COCO dataset, 80 classes)
4. Visualization — results appear with color-coded boxes (🔴→🟡→🟢 based on confidence)
5. Export — download CSV reports or annotated images

Stack: FastAPI + YOLOv8 + OpenCV + vanilla HTML/CSS/JS.

👤 Who made it and why?

Created by L2santos29 as a hands-on project to explore computer vision, containerized deployment, and full-stack development. The goal was to build a practical CV + LLM orchestration service that is easy to use, deployable with Docker, and extensible for future AI features.
